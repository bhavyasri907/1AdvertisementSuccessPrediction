# Advertisement Success Prediction System

An AI-powered system that predicts advertisement success using machine learning and computer vision.

## Features

- 🤖 **ML Predictions**: Predicts ad ratings, success probability, and money-back guarantee
- 👁️ **Video Analysis**: Analyzes uploaded videos for visual metrics
- 📊 **Interactive Dashboard**: Streamlit-based user interface
- 🔌 **REST API**: FastAPI backend for programmatic access

# Clone or download the project
cd advertisement-success-predictor

# Run the main script
